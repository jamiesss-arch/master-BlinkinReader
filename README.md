Blinkin' Reader is a C project I created with the goal of allowing people who are both blind and deaf to "read". By teaching those patients the Morse code, we can then vibrate text via a bracelet, allowing those people a similar experience to reading a book. The vibrator component was replaced by a LED for an easier understanding of the concept.
